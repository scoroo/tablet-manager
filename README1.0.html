🔧 Detailed Installation

Step 1: Install Node.js

Download from: https://nodejs.org/
Choose LTS version (18.x or 20.x)
Install with default settings

Step 2: Extract and Install Dependencies

# Extract the tablet-manager folder
cd tablet-manager
npm install

Step 3: Enable ADB over WiFi (One Time Per Tablet)

# Connect tablet via USB first
# Then run the batch file:
enable-wifi-adb.bat

# Or for multiple tablets at once:
enable-all-wifi.bat

Step 4: Start the Server

# Double-click start.bat OR run:
node server.js

# Open browser to:
http://localhost:3000

🔧 Troubleshooting

ProblemSolution
Tablet not showing in listWait 30 seconds or click "Scan Now" button
Detect shows "Failed"Run enable-wifi-adb.bat with USB connected first
"Device offline" messageMake sure tablet is on same WiFi network as PC
Connection refusedRun adb tcpip 5555 again with USB connected
Installation failsCheck APK compatibility and tablet storage space
